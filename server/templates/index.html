How grading works

Agents see only the task pages below — rewards are computed server-side from ground-truth state, never scraped from the DOM. Drive an episode from any HTTP client:

curl -X POST {host}/api/email-triage/reset -H 'Content-Type: application/json' \
  -d '{"seed": 42, "latency_ms": 800, "layout_shift": true}'
# -> {"instruction": "In the inbox, archive every email from ..."}
#    ...point your agent at /task/email-triage ...
curl {host}/api/email-triage/verify
# -> {"success": false, "subgoals": {"archived_all_from_target": true, ...}}
